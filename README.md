# Predicting-Customer-Purchase-Intention-Using-Supervised-Learning.

This project predicts whether a user will make a purchase during an online shopping session.

## Dataset
https://archive.ics.uci.edu/ml/datasets/Online+Shoppers+Purchasing+Intention+Dataset

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- KNN

## Results
- Random Forest performed best
- KNN also performed well

## How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Open notebook and run all cells
